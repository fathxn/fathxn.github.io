Akhirnya deploy juga ke Github

Sebenarnya sudah dari kemarin-kemarin ingin deploy blog ini ke GitHub tapi karena merasa masih kurang puas dengan segala utak-atik yang saya lakukan terhadap blog ini jadi tertunda deh. Tapi disisi lain juga sudah bingung apalagi yang mau di utak-atik, akhirnya sempat saya tinggalkan blog ini untuk beberapa hari karena pekerjaan juga.

Setelah beberapa hari berlalu dengan kesibukan, akhirnya yasudah lah deploy aja dulu daripada tidak tau dan bingung mau diapakan lagi blog ini yakan …

Cari tutorial sana sini cara deploy atau host Hugo ke GitHub dan ketemulah tutorialnya dengan step-by-step yang clear dan mudah dipahami oleh saya yang masih awam dan membuat blog Hugo ini ter-deploy dengan lancar ke GitHub.

Ada pun step-by-step nya berikut ini:

  1. Pertama bikin repository dulu di akun GitHub dengan nama username.github.io, ini akan membuat repository sekaligus untuk GitHub pages kamu nanti yang dapat dikunjungi, jangan lupa ganti username dengan username GitHub kamu dan jangan centang README.md
  2. Build blog Hugo nya dulu dengan perintah hugo -v
  3. Akan ada folder baru bernama public dan masuk ke folder tersebut dengan perintah cd public
  4. Kemudian ketik perintah git init
  5. Lalu ketik perintah git remote add origin https://github.com/username/username.github.io.git, jangan lupa ganti username dengan username GitHub kamu

  7. Sudah deh tinggal tunggu beberapa menit dan buka deh GitHub pages kamu di username.github.io
